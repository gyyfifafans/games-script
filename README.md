## 使用方法

本软件依赖WebDriverAgent及其Python Binding, Opencv-Python Binding, 请自行搜索安装。具体安装方法请参见[这里](https://github.com/wangshub/wechat_jump_game/wiki/Android-和-iOS-操作步骤#二ios-手机操作步骤)

安装好依赖后运行

```
python3 start.py
```

## 软件原理

本软件通过截图与图片库中的图片比对判断游戏状态并操作。
